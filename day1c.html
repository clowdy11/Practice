Semantic Markups

We saw the structural markup, elements that describe both heading and paragraphs and affect the structure of the page. Now it's time to learn about semantic markup which provides extra information to the content of the page

First one. The use of the element <strong> indicates that its content has strong importance. By default, browser will show the contents inside the <strong> tag in bold.

Second one.The <em> element indicates emphasis. By default the content inside will be shown in italic.

Quotations

The <blockquote> element is used for longer quotes that takes an entire paragraph. Note how the <p> element is still used inside the <blockquote> element. Important. Although browsers tend to indent the content inside the <blockquote> tags, it is important to not use this tag for that intention.

The <q> element is used for shorter quotes. It is supposed to put quotation marks around, but yunno Internet Explorer doesn't... So many people avoid the use of this tag.

Both elements may use the "cite" attribute to indicate where the quote is from. Its valued should be an URL.
